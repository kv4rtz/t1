Получение всех постов
Нужно выполнить GET запрос по /posts
Ожидаемый результат: получить список постов (пустой либо с постами)


Получение одного поста по заголовку
Нужно выполнить GET запрос по /posts/{title}
Ожидаемый результат: вернется нужный пост, 404 (NotFound) если пост не найден.


Создание поста
Нужно выполнить POST запрос по /posts/{title}/{content}
Ожидаемый результат: создасться пост с указанным title и content.
Проверка: запросить пост по GET запросу /posts/{title}


Редактирование поста (изменение title)
Нужно выполнить PATCH запрос по /posts/{oldTitle}/{newTitle}
Ожидаемый результат: обновится title у искомого поста.
Проверка: запросить пост по GET запросу /posts/{newTitle} после обновления


Удалить пост
Нужно выполнить DELETE запрос по /posts/{title}
Ожидаемый результат: пост удалится.
Проверка: запросить все посты по GET запросу /posts


Получение комментариев поста
Нужно выполнить GET запрос по /posts/{title}/comments
Ожидаемый результат: получить список комментариев поста, 404 (NotFound) если пост не найден.


Создание комментария к посту
Нужно выполнить POST запрос по /posts/{title}/comments/{text}
Ожидаемый результат: создасться комментарий и вернется commentId.
Проверка: запросить комментарии по GET запросу /posts/{title}/comments и убедиться что комментарий присутствует


Редактирование комментария
Нужно выполнить PATCH запрос по /posts/{title}/comments/{commentId}/{newText}
Ожидаемый результат: обновится текст комментария, 404 (NotFound) если комментарий не найден.
Проверка: запросить комментарии по GET запросу /posts/{title}/comments после обновления


Удалить комментарий
Нужно выполнить DELETE запрос по /posts/{title}/comments/{commentId}
Ожидаемый результат: комментарий удалится, 404 (NotFound) если комментарий не найден.
Проверка: запросить комментарии по GET запросу /posts/{title}/comments после удаления
