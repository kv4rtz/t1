Получение всех событий
Нужно выполнить GET запрос по /events
Ожидаемый результат: получить список событий (пустой либо с событиями)


Получение одного события по названию
Нужно выполнить GET запрос по /events/{title}
Ожидаемый результат: вернется нужное событие, 404 (NotFound) если событие не найдено.


Создание события
Нужно выполнить POST запрос по /events/{title}/{when}
Ожидаемый результат: создасться событие с указанной датой/временем (ISO формат, пример: 2026-01-01T10:00:00).
Проверка: запросить событие по GET запросу /events/{title}


Редактирование события (изменение title)
Нужно выполнить PATCH запрос по /events/{oldTitle}/{newTitle}
Ожидаемый результат: обновится title у события.
Проверка: запросить событие по GET запросу /events/{newTitle}


Редактирование времени события
Нужно выполнить PATCH запрос по /events/time/{title}/{newWhen}
Ожидаемый результат: обновится поле when у события.
Проверка: запросить событие по GET запросу /events/{title}


Удаление события
Нужно выполнить DELETE запрос по /events/{title}
Ожидаемый результат: событие удалится.
Проверка: запросить все события по GET запросу /events


Уведомления о предстоящих событиях
Нужно выполнить GET запрос по /notifications/{now}/{windowMinutes}
Ожидаемый результат: вернется список событий, которые попадают в интервал [now, now + windowMinutes].
Пример: GET /notifications/2026-01-01T10:00:00/30 вернет события между 10:00:00 и 10:30:00.
