Получение всех опросов
Нужно выполнить GET запрос по /polls
Ожидаемый результат: получить список опросов (пустой либо с опросами)


Получение одного опроса по названию
Нужно выполнить GET запрос по /polls/{title}
Ожидаемый результат: вернется нужный опрос, 404 (NotFound) если опрос не найден.


Создание опроса
Нужно выполнить POST запрос по /polls/{title}
Ожидаемый результат: создасться опрос.
Проверка: запросить опрос по GET запросу /polls/{title}


Добавление варианта ответа
Нужно выполнить POST запрос по /polls/{title}/options/{optionName}
Ожидаемый результат: добавится вариант ответа в опрос, 404 (NotFound) если опрос не найден.
Проверка: запросить опрос по GET запросу /polls/{title}


Голосование
Нужно выполнить POST запрос по /polls/{title}/vote/{optionName}
Ожидаемый результат: увеличится счетчик голосов у выбранного варианта, 404 (NotFound) если опрос или вариант не найден.


Получение результатов опроса
Нужно выполнить GET запрос по /polls/{title}/results
Ожидаемый результат: вернется словарь {вариант: количествоГолосов}, 404 (NotFound) если опрос не найден.


Удаление опроса
Нужно выполнить DELETE запрос по /polls/{title}
Ожидаемый результат: опрос удалится.
Проверка: запросить все опросы по GET запросу /polls
