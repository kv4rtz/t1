Получение всех задач из списка
Нужно выполнить GET запрос по /tasks
Ожидаемый результат: получить список задач (пустой либо с задачами)


Получение одной задачи по имени
Нужно выполнить GET запрос по /tasks/{имя}
Ожидаемый результат: вернется нужная задача, 404 (NotFound) если задача не найдена.


Создание задачи в список
Нужно выполнить POST запрос по /tasks/{имя}
Ожидаемый результат: создасться задача с completed = false.
Проверка: запросить задачу по GET запросу /tasks/{имя}


Редактирование задачи по имени
Нужно выполнить PATCH запрос по /tasks/{имя}/{новоеИмя}
Ожидаемый результат: обновится имя у искомой задачи
Проверка: запросить задачу по GET запросу /tasks/{новоеИмя} после обновления


Пометить задачу как выполненную
Нужно выполнить PATCH запрос по /tasks/complete/{имя}
Ожидаемый результат: обновится поле completed на true у искомой задачи
Проверка: запросить задачу по GET запросу /tasks/{имя} после обновления


Удалить задачу из списка
Нужно выполнить DELETE запрос по /tasks/{имя}
Ожидаемый результат: задача удалится из списка
Проверка: запросить все задачи по GET запросу /tasks
