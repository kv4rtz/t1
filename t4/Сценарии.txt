Получение всех книг
Нужно выполнить GET запрос по /books
Ожидаемый результат: получить список книг (пустой либо с книгами)


Получение одной книги по названию
Нужно выполнить GET запрос по /books/{title}
Ожидаемый результат: вернется нужная книга, 404 (NotFound) если книга не найдена.


Добавление книги
Нужно выполнить POST запрос по /books/{title}/{author}
Ожидаемый результат: создасться книга с borrowed = false и borrowedBy = null.
Проверка: запросить книгу по GET запросу /books/{title}


Удаление книги
Нужно выполнить DELETE запрос по /books/{title}
Ожидаемый результат: книга удалится.
Проверка: запросить все книги по GET запросу /books


Получение всех читателей
Нужно выполнить GET запрос по /readers
Ожидаемый результат: получить список читателей (пустой либо с читателями)


Регистрация читателя
Нужно выполнить POST запрос по /readers/{name}
Ожидаемый результат: создасться читатель.
Проверка: запросить всех читателей по GET запросу /readers


Выдача книги читателю
Нужно выполнить POST запрос по /borrow/{title}/{readerName}
Ожидаемый результат: у книги выставится borrowed = true и borrowedBy = readerName.
Проверка: запросить книгу по GET запросу /books/{title}


Возврат книги читателем
Нужно выполнить POST запрос по /return/{title}/{readerName}
Ожидаемый результат: у книги выставится borrowed = false и borrowedBy = null.
Проверка: запросить книгу по GET запросу /books/{title}


Ошибки выдачи/возврата
Нужно выполнить POST запрос по /borrow/{title}/{readerName}
Ожидаемый результат: 404 (NotFound) если книга или читатель не найдены, 400 (BadRequest) если книга уже выдана.

Нужно выполнить POST запрос по /return/{title}/{readerName}
Ожидаемый результат: 404 (NotFound) если книга или читатель не найдены, 400 (BadRequest) если книга не была выдана или выдана другому читателю.
